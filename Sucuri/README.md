# Features
 - Themes
 - Lots of commands
 - Very customizable
 - Very fast Nitro sniper

# Usage
 1. Download Sucuri
 2. Enter token in config.json file
 3. Launch the selfbot by clicking twice on **start.bat**
 4. Wait untill token is verified
 5. Open help menu using this command: **./help**
